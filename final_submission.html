1. Executive Summary and Problem Statement

The residential and commercial solar distribution industry faces operational inefficiencies at the front-end sales and engineering funnel. Specifically, distributors face four primary friction points:

  • Unstructured Customer Discovery: Leads arrive via messy customer emails or chat transcripts. Ingesting this data manually is slow and prone to errors.
  • Technical Sizing Misalignment: Sales representatives frequently quote system sizes that physically cannot fit on the customer's roof footprint, leading to engineering design loops and re-quoting.
  • Supply Chain Disconnect: Sourcing out-of-stock components causes delays. We solve this by implementing real-time warehouse inventory checks in the BOM pipeline.
  • Lifecycle ROI Financials: Industry calculators assume zero utility inflation and zero panel degradation. Our engine compounds grid inflation at 5.0% and applies a precise variable degradation curve (0.5% in Year 1, 0.25% in subsequent years).

The Solution: Our Multi-Agent System automates the pipeline from raw query to a custom compounding sales proposal. It integrates stateful memory state persistence with deterministic calculations and multi-tier guardrails (including location/address verification).

2. Agent Architecture

The system leverages an Orchestrator-Router Pattern with a shared state memory bank:

Pipeline Workflow

[Raw Ingestion] -> TriageAgent (Extracts parameters)
↳ Pre-Flight Guardrails Check (Invalid Location, Capacity Cap, Negative Space)
    ↳ TechnicalAgent (Vets physical roof area constraints)
        ↳ Post-Tech Guardrails (ROOF_SPACE_EXCEEDED)
            ↳ FinancialLogisticsAgent (Selects Inverter Type, Sourcing stock, Variable ROI calculation)
                ↳ Outputs: Bill of Materials and Compounding ROI Proposal

3. Mathematics and Deterministic Calculations

Upgraded Footprint Sizer (625W Panel)

We model high-power 625W panels with a footprint of 24.0 sq ft per panel. The system sizer executes:

Panels Needed = ceil(Target kW / 0.625 kW)
Required Footprint (sq ft) = Panels Needed * 24.0

Variable Degradation Lifecycle ROI Model

Grid utility rates compound against 5.0% grid inflation. Solar generation degrades dynamically (0.5% Year 1, 0.25% in subsequent years):

Year 1 Gen = Capacity * 1500 kWh
Year y Gen (y >= 2) = (Capacity * 1500) * (1 - 0.005) * (1 - 0.0025)^(y-2)

Dynamic Solar/Micro Inverters Selector

The system dynamically determines the optimal inverter technology based on target system capacity:

  • Decentralized Microinverters (1 per panel): Chosen for systems < 10 kW.
  • Central String Solar Inverters (1 per system): Chosen for larger or commercial systems (≥ 10 kW).

def calculate_required_footprint(target_kw: float, panel_efficiency: float = 0.20) -> dict:
    """
    Calculate the required physical footprint (sq. ft.) based on target kW capacity.
    Standard panel specs: 625W (0.625 kW) power output, dimensions ~24 sq. ft.
    """
    if target_kw <= 0:
        raise ValueError("Target capacity must be greater than 0 kW.")
        
    panel_wattage_kw = 0.625  # 625W
    panel_area_sqft = 24.0    # 24 sq ft
    
    # Calculate number of panels needed
    panels_needed = math.ceil(target_kw / panel_wattage_kw)
    total_footprint_sqft = panels_needed * panel_area_sqft
    
    return {
        "target_kw": target_kw,
        "panels_needed": panels_needed,
        "panel_wattage_w": 625,
        "panel_area_sqft": panel_area_sqft,
        "required_footprint_sqft": total_footprint_sqft,
        "efficiency_used": panel_efficiency
    }

def calculate_roi(
    kw_capacity: float, 
    initial_investment: float, 
    utility_rate: float, 
    annual_usage_kwh: float, 
    escalation_rate: float = 0.05, 
    degradation_rate_y1: float = 0.005,
    degradation_rate_subsequent: float = 0.0025
) -> dict:
    """
    Calculate 25-year lifecycle savings and payback period.
    Assumes standard insolation: 1 kW of solar capacity generates 1,500 kWh of energy per year.
    Escalates grid energy costs annually (escalation_rate).
    Degrades solar production: 0.5% in 1st year (applied at year 2), 0.25% in upcoming years (year 3+).
    """
    if kw_capacity <= 0 or initial_investment <= 0 or utility_rate <= 0 or annual_usage_kwh <= 0:
        raise ValueError("All input parameters must be positive non-zero values.")
        
    annual_generation_base = kw_capacity * 1500.0
    timeline = []
    cumulative_savings = 0.0
    payback_year = None
    
    for year in range(1, 26):
        # Calculate degradation for this year
        if year == 1:
            generation_this_year = annual_generation_base
        else:
            generation_this_year = annual_generation_base * (1.0 - degradation_rate_y1) * ((1.0 - degradation_rate_subsequent) ** (year - 2))
            
        # Escalate grid rates
        rate_this_year = utility_rate * ((1.0 + escalation_rate) ** (year - 1))
        
        # Calculate utility cost offsets
        offset_kwh = min(generation_this_year, annual_usage_kwh)
        export_kwh = max(0.0, generation_this_year - annual_usage_kwh)
        
        savings_from_offset = offset_kwh * rate_this_year
        savings_from_export = export_kwh * (rate_this_year * 0.5)
        annual_savings = savings_from_offset + savings_from_export
        
        cumulative_savings += annual_savings
        net_position = cumulative_savings - initial_investment
        
        # Track when we break even
        if net_position >= 0 and payback_year is None:
            prev_cumulative = cumulative_savings - annual_savings
            needed = initial_investment - prev_cumulative
            fraction = needed / annual_savings if annual_savings > 0 else 0.0
            payback_year = round((year - 1) + fraction, 2)
            
        timeline.append({
            "year": year,
            "solar_gen_kwh": round(generation_this_year, 1),
            "utility_rate_per_kwh": round(rate_this_year, 4),
            "annual_savings_usd": round(annual_savings, 2),
            "cumulative_savings_usd": round(cumulative_savings, 2),
            "net_position_usd": round(net_position, 2)
        })
        
    net_25yr_savings = cumulative_savings - initial_investment
    net_roi_percent = (cumulative_savings / initial_investment) * 100.0 if initial_investment > 0 else 0.0
    
    return {
        "initial_investment_usd": initial_investment,
        "annual_usage_kwh": annual_usage_kwh,
        "base_annual_generation_kwh": annual_generation_base,
        "escalation_rate": escalation_rate,
        "degradation_rate_y1": degradation_rate_y1,
        "degradation_rate_subsequent": degradation_rate_subsequent,
        "payback_period_years": payback_year if payback_year is not None else "Never (>25 yrs)",
        "total_25yr_savings_usd": round(cumulative_savings, 2),
        "net_25yr_savings_usd": round(net_25yr_savings, 2),
        "roi_percentage": round(net_roi_percent, 2),
        "timeline": timeline
    }

def check_inventory_and_bom(target_kw: float, inverter_preference: str = "auto") -> dict:
    """
    Cross-reference required quantities with warehouse stocks.
    Handles Solar Inverters / Microinverters central vs distributed logic.
    """
    if target_kw <= 0:
        raise ValueError("Target capacity must be greater than 0 kW.")
        
    panels_needed = math.ceil(target_kw / 0.625)
    racking_needed = panels_needed
    bos_needed = 1
    
    # Determine Inverter Type
    if inverter_preference == "micro" or (inverter_preference == "auto" and target_kw < 10.0):
        inverter_type = "microinverter"
        inverters_needed = panels_needed
        inverter_db_key = "microinverters"
        inverter_display_name = INVENTORY_DB["microinverters"]["name"]
    else:
        inverter_type = "string_inverter"
        inverters_needed = 1  
        inverter_db_key = "string_inverters"
        inverter_display_name = INVENTORY_DB["string_inverters"]["name"]
        
    bom = []
    shortage_detected = False
    shortages = []
    
    # Check Panels
    panels_stock = INVENTORY_DB["panels"]["stock"]
    panels_cost = panels_needed * INVENTORY_DB["panels"]["unit_cost"]
    panels_status = "Available"
    if panels_needed > panels_stock:
        panels_status = f"Shortage (Needs {panels_needed}, Only {panels_stock} in stock)"
        shortage_detected = True
        shortages.append(f"Solar Panels (Short by {panels_needed - panels_stock})")
    bom.append({
        "item": "Panels",
        "name": INVENTORY_DB["panels"]["name"],
        "required": panels_needed,
        "in_stock": panels_stock,
        "unit_cost_usd": INVENTORY_DB["panels"]["unit_cost"],
        "total_cost_usd": panels_cost,
        "status": panels_status
    })
    
    # Check Inverters
    inverters_stock = INVENTORY_DB[inverter_db_key]["stock"]
    inverters_cost = inverters_needed * INVENTORY_DB[inverter_db_key]["unit_cost"]
    inverters_status = "Available"
    if inverters_needed > inverters_stock:
        inverters_status = f"Shortage (Needs {inverters_needed}, Only {inverters_stock} in stock)"
        shortage_detected = True
        shortages.append(f"Inverters (Short by {inverters_needed - inverters_stock})")
    bom.append({
        "item": "Solar Inverters / Microinverters",
        "name": inverter_display_name,
        "required": inverters_needed,
        "in_stock": inverters_stock,
        "unit_cost_usd": INVENTORY_DB[inverter_db_key]["unit_cost"],
        "total_cost_usd": inverters_cost,
        "status": inverters_status
    })
    
    # Check Racking
    racking_stock = INVENTORY_DB["racking"]["stock"]
    racking_cost = racking_needed * INVENTORY_DB["racking"]["unit_cost"]
    racking_status = "Available"
    if racking_needed > racking_stock:
        racking_status = f"Shortage (Needs {racking_needed}, Only {racking_stock} in stock)"
        shortage_detected = True
        shortages.append(f"Racking kits (Short by {racking_needed - racking_stock})")
    bom.append({
        "item": "Racking",
        "name": INVENTORY_DB["racking"]["name"],
        "required": racking_needed,
        "in_stock": racking_stock,
        "unit_cost_usd": INVENTORY_DB["racking"]["unit_cost"],
        "total_cost_usd": racking_cost,
        "status": racking_status
    })
    
    # Check Balance of System (BOS)
    bos_stock = INVENTORY_DB["bos"]["stock"]
    bos_cost = bos_needed * INVENTORY_DB["bos"]["unit_cost"]
    bos_status = "Available"
    if bos_needed > bos_stock:
        bos_status = f"Shortage (Needs {bos_needed}, Only {bos_stock} in stock)"
        shortage_detected = True
        shortages.append(f"BOS kits (Short by {bos_needed - bos_stock})")
    bom.append({
        "item": "BOS",
        "name": INVENTORY_DB["bos"]["name"],
        "required": bos_needed,
        "in_stock": bos_stock,
        "unit_cost_usd": INVENTORY_DB["bos"]["unit_cost"],
        "total_cost_usd": bos_cost,
        "status": bos_status
    })
    
    hardware_cost = panels_cost + inverters_cost + racking_cost + bos_cost
    labor_cost = 1200.00 + (150.00 * panels_needed)
    permit_fees = 800.00
    subtotal = hardware_cost + labor_cost + permit_fees
    distributor_markup = subtotal * 0.30
    total_retail_quote = subtotal + distributor_markup
    
    return {
        "target_kw": target_kw,
        "bom": bom,
        "hardware_cost_usd": round(hardware_cost, 2),
        "labor_cost_usd": round(labor_cost, 2),
        "permits_and_engineering_usd": round(permit_fees, 2),
        "subtotal_cost_usd": round(subtotal, 2),
        "markup_usd": round(distributor_markup, 2),
        "total_retail_quote_usd": round(total_retail_quote, 2),
        "shortage_detected": shortage_detected,
        "shortages": shortages
    }

class SolarTriageSchema(BaseModel):
    system_size_kw: Optional[float] = Field(None, description="Target capacity in kW.")
    budget_usd: Optional[float] = Field(None, description="Customer budget.")
    location: Optional[str] = Field(None, description="Valid address or city/state.")
    utility_rate_kwh: Optional[float] = Field(None, description="Utility rate per kWh.")
    annual_usage_kwh: Optional[float] = Field(None, description="Annual energy usage.")
    roof_space_sqft: Optional[float] = Field(None, description="Available roof space.")
    notes: Optional[str] = Field(None, description="General preferences.")

class InMemoryContextBank:
    def __init__(self):
        self.session_id: str = "session_001"
        self.raw_query: str = ""
        self.triage_data: Dict[str, Any] = {}
        self.technical_data: Dict[str, Any] = {}
        self.financial_logistics_data: Dict[str, Any] = {}
        self.history: List[Dict[str, str]] = []
        self.validation_errors: List[str] = []
        self.status: str = "INIT"

    def add_to_history(self, role: str, content: str):
        self.history.append({"role": role, "content": content})

    def update_triage(self, data: Dict[str, Any]):
        for k, v in data.items():
            if v is not None:
                self.triage_data[k] = v

    def clear_errors(self):
        self.validation_errors = []

    def add_error(self, err: str):
        if err not in self.validation_errors:
            self.validation_errors.append(err)

class TriageAgent:
    def __init__(self, client: Optional[Any] = None):
        self.client = client

    def process(self, query: str, context: InMemoryContextBank) -> Dict[str, Any]:
        extracted = {}
        if SDK_AVAILABLE and self.client and os.environ.get("GEMINI_API_KEY"):
            try:
                prompt = f"Analyze query and extract: system_size_kw, budget_usd, location, utility_rate_kwh, annual_usage_kwh, roof_space_sqft. Query: {query}"
                response = self.client.models.generate_content(
                    model="gemini-2.5-flash",
                    contents=prompt,
                    config=types.GenerateContentConfig(
                        response_mime_type="application/json",
                        response_schema=SolarTriageSchema,
                        system_instruction="You are an expert sales ingestion agent."
                    )
                )
                extracted = json.loads(response.text)
            except Exception:
                extracted = self._regex_fallback(query)
        else:
            extracted = self._regex_fallback(query)
        context.update_triage(extracted)
        context.status = "TRIAGED"
        return extracted

    def _regex_fallback(self, query: str) -> Dict[str, Any]:
        data = {"system_size_kw": None, "budget_usd": None, "location": None, "utility_rate_kwh": None, "annual_usage_kwh": None, "roof_space_sqft": None}
        loc_match = re.search(r"in\s+([A-Za-z\s,.-]{3,30})(?=\.|\bin\b|\bwith\b|\band\b|\bmy\b|\bfor\b)", query, re.IGNORECASE)
        if loc_match:
            loc = loc_match.group(1).strip()
            if loc.endswith(".") or loc.endswith(","):
                loc = loc[:-1].strip()
            data["location"] = loc
        # Capacity
        kw_match = re.search(r"(-?\d+(\.\d+)?)\s*k[wW](?![hH])", query)
        if kw_match: data["system_size_kw"] = float(kw_match.group(1))
        # Rate
        rate_match = re.search(r"(?:pay|rate)?\s*\$\s*(0\.\d+)\s*/\s*k[wW]h", query, re.IGNORECASE)
        if rate_match: data["utility_rate_kwh"] = float(rate_match.group(1))
        # Budget
        all_dollars = re.finditer(r"\$\s*(-?\d{1,3}(?:,\d{3})*(?:\.\d+)?)", query)
        for match in all_dollars:
            val = float(match.group(1).replace(",", ""))
            if val > 5.0: data["budget_usd"] = val
        # Usage
        usage_match = re.search(r"(\d{1,3}(,\d{3})*)\s*k[wW]h\s*(?:annually|annual|year)", query, re.IGNORECASE)
        if usage_match: data["annual_usage_kwh"] = float(usage_match.group(1).replace(",", ""))
        # Roof Space
        roof_match = re.search(r"(-?\d{1,5}(,\d{3})*)\s*(?:sq\s*ft|square\s*feet)", query, re.IGNORECASE)
        if roof_match: data["roof_space_sqft"] = float(roof_match.group(1).replace(",", ""))
        return data

class TechnicalAgent:
    def __init__(self, client: Optional[Any] = None):
        self.client = client

    def process(self, context: InMemoryContextBank) -> Dict[str, Any]:
        triage = context.triage_data
        target_kw = triage.get("system_size_kw", 5.0)
        roof_space = triage.get("roof_space_sqft")
        
        footprint_res = solar_tools.calculate_required_footprint(target_kw)
        required_sqft = footprint_res["required_footprint_sqft"]
        panels_needed = footprint_res["panels_needed"]
        
        valid = True
        if roof_space is not None and required_sqft > roof_space:
            valid = False
            context.add_error("ROOF_SPACE_EXCEEDED")
            
        tech_data = {"required_footprint_sqft": required_sqft, "panels_needed": panels_needed, "panel_wattage_w": 625, "constraints_valid": valid, "technical_notes": "fits" if valid else "exceeded"}
        context.technical_data = tech_data
        return tech_data

class FinancialLogisticsAgent:
    def __init__(self, client: Optional[Any] = None):
        self.client = client

    def process(self, context: InMemoryContextBank) -> Dict[str, Any]:
        triage = context.triage_data
        target_kw = triage.get("system_size_kw", 5.0)
        utility_rate = triage.get("utility_rate_kwh", 0.23)
        annual_usage = triage.get("annual_usage_kwh", 10000.0)
        
        bom_res = solar_tools.check_inventory_and_bom(target_kw)
        initial_investment = bom_res["total_retail_quote_usd"]
        
        roi_res = solar_tools.calculate_roi(target_kw, initial_investment, utility_rate, annual_usage)
        if bom_res["shortage_detected"]:
            context.add_error("WAREHOUSE_SHORTAGE")
            
        fin_log = {"bom_details": bom_res, "roi_details": roi_res, "quote_summary_usd": initial_investment, "payback_period_years": roi_res["payback_period_years"], "net_25yr_savings_usd": roi_res["net_25yr_savings_usd"], "roi_percentage": roi_res["roi_percentage"]}
        context.financial_logistics_data = fin_log
        
        proposal = f"""### Sizing Summary
Capacity: {target_kw:.2f} kW. Investment: ${initial_investment:,.2f}. 25-Yr Savings: ${roi_res['net_25yr_savings_usd']:,.2f}."""
        fin_log["proposal_narrative"] = proposal
        context.status = "COMPLETE" if not context.validation_errors else "ERROR"
        return fin_log

class SolarOrchestrator:
    def __init__(self):
        self.context = InMemoryContextBank()
        self.triage_agent = TriageAgent()
        self.technical_agent = TechnicalAgent()
        self.fin_agent = FinancialLogisticsAgent()

    def receive_query(self, user_query: str) -> str:
        self.context.raw_query = user_query
        self.context.clear_errors()
        self.triage_agent.process(user_query, self.context)
        self._run_pre_flight_guardrails()
        if self.context.validation_errors: return "Halted on guardrails"
        self.technical_agent.process(self.context)
        self.fin_agent.process(self.context)
        return "Complete"

    def _check_real_location(self, location: str) -> bool:
        if not location: return False
        loc = location.strip().lower()
        if loc in ["unknown", "none", "null", "undefined"]: return False
        if any(w in loc for w in ["house", "roof", "solar"]): return False
        return len(loc) >= 3

    def _run_pre_flight_guardrails(self):
        triage = self.context.triage_data
        location = triage.get("location")
        if location is not None and location != "":
            loc_lower = location.strip().lower()
            if loc_lower not in ["unknown", "none", "null", "undefined"]:
                if not self._check_real_location(location):
                    self.context.add_error("INVALID_LOCATION")
        kw = triage.get("system_size_kw")
        if kw and kw > 93.75:
            self.context.add_error("CAPACITY_EXCEEDS_WAREHOUSE_LIMIT")
